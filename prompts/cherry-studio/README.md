# Cherry Studio Prompts

这个目录放的是可直接粘贴到 Cherry Studio 的提示词模板，默认面向本仓库提供的 `wedecode` MCP。

## 文件

- `security-audit.md`
  面向小程序安全审计，重点看权限、敏感接口、认证、存储、请求、硬编码密钥、前端注入面。
- `reverse-analysis.md`
  面向功能逆向和结构分析，重点看路由、分包、核心页面、请求链路、登录支付、插件与云开发。

## 使用方式

1. 在 Cherry Studio 中添加并启用 `wedecode` MCP。
2. 新建一个助手，或打开已有助手的系统提示词配置。
3. 把对应文件内容完整粘贴进去。
4. 聊天时先让模型扫描本机小程序，再按编号选择目标。

## 推荐起手式

```text
扫描我本机的微信小程序，列出前 10 个给我选。
```

选中后再说：

```text
分析第 3 个。
```

如果你已经知道本地 `wxapkg` 路径，也可以直接说：

```text
直接反编译这个 wxapkg，并按当前助手的分析目标继续：
C:\path\to\__APP__.wxapkg
```
